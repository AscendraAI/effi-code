# super-agent

**클로드를 메인 오케스트라로 두고, 라우팅·캐싱·캐스케이드로 비용을 최소화하고, 유료 쿼터가 바닥나면 로컬 모델이 이어받는 — 파일 기반 코딩 에이전트 오케스트레이션.**

> 2026년 최신 에이전트 연구(Anthropic 멀티에이전트·arXiv·Cognition·라우팅/캐싱 실측)를 근거로 설계.
> 설계 근거 요약: `docs/why.md`

---

## 다운받으면 바로 쓸 수 있나?

정직하게 — **부분적으로.**

| 부분 | 바로? |
|---|---|
| 오케스트레이션 규칙·템플릿 (`ORCHESTRATION.md`, `templates/`) | ✅ 즉시 — 프로젝트에 넣고 AI에 물리면 끝 |
| 비용 절감(라우팅·캐싱) + 로컬 폴백 | ⚙️ **1회 세팅 필요** (~10~15분) — `SETUP.md` / `setup.sh` |

"제로 설정 원클릭"은 아닙니다. **`./setup.sh` 한 번 + 안내 따라가기**면 작동합니다.

## 설계 한 줄

> **성능은 강한 단일 클로드 스레드**에서, **비용은 라우팅·캐싱·캐스케이드**에서, **안전망은 쿼터 소진 시 로컬 폴백**에서.
> 멀티에이전트는 코딩에선 아껴 쓴다 (증거: 동일 예산이면 단일 에이전트가 멀티를 매칭/능가).

## 왜 이렇게 (핵심 근거)

- **클로드 메인 오케스트라는 옳다** — Anthropic 실측: 강모델 리드 + 싼 서브가 단일 강모델을 90.2% 능가. 단 그건 *리서치* 태스크.
- **코딩에선 멀티에이전트 절제** — 토큰 15배, 대부분 코딩엔 부적합. 강 단일 스레드 + *진짜 병렬 가능한 일*(리뷰·독립 모듈)에만 서브에이전트.
- **비용 1순위 = 라우팅** — 클로드코드 토큰 85%는 최상위 모델 불필요. 어려운 것만 위로. 실사용 62% 절감.
- **캐싱 함정** — 여러 프로바이더로 대화를 쪼개면 클로드 캐시가 깨져 10배. 메인 스레드는 캐시 연속 유지.
- **로컬 = 백스톱** — 일상 코딩 80% 커버, 최난도는 못 함, 2~10배 느림. 쿼터 소진 시 이어받기 딱 좋음.

## 구성

```
super-agent/
├── README.md
├── SETUP.md              # 세팅 (하네스 + 로컬 LLM + 라우터/폴백)
├── setup.sh              # 자동 설치 스크립트
├── ORCHESTRATION.md      # 운영 규칙 (강 단일 스레드 + 선택적 서브 + 검증 게이트)
├── ROUTING.md            # 캐스케이드 라우팅 + 비용 규율 (캐싱 함정 포함)
├── FALLBACK.md           # 쿼터 소진 → 로컬 폴백 설정
├── LOCAL-MODELS.md       # 맥 RAM별 모델 + Ollama 세팅
├── bin/sag                # 런처/토글 (구독 Claude ↔ 로컬 폴백)
├── bin/sag-run            # 로컬 워커 — 지루한 대량 작업을 로컬에 위임
├── docs/why.md           # 설계 근거 (딥리서치 요약)
└── templates/            # task · brief · log
```

## 빠른 시작
1. `git clone … && cd super-agent`
2. `./setup.sh` (Ollama 설치 → 맥 RAM 감지 → 로컬 모델 다운)
3. `ln -s "$PWD/bin/sag" /opt/homebrew/bin/sag` (폴백 토글을 PATH에)
4. `ORCHESTRATION.md`를 코딩 세션에 규칙으로 물린다
5. 평소 `sag`(구독 Claude) → 한도 소진 시 `sag local`(무료 로컬)

## 폴백 토글 (구독 사용자)
구독은 프록시로 못 돌린다(Anthropic ToS) → 라우터 대신 **토글**:
```bash
sag            # 평소 = 구독 Claude+Codex+Gemini
sag local      # 셋 다 한도 소진 시 → 로컬(무료, 16GB=qwen2.5-coder:7b)
sag status     # 상태
```
자세히: `FALLBACK.md`.
